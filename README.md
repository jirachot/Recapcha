# recaptcha2-ci
Google Recaptcha v2 with Codeigniter 3

## Demo
> http://demo.momo.my.id/recaptcha2_ci/

change
```
$config['recaptcha_sitekey']   = "";
$config['recaptcha_secretkey'] = "";
$config['lang']                = "";
```

on application->config->recaptcha.php with your key
